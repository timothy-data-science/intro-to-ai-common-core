# Common Core: Introduction to AI & Data Fluency

## Purpose
This repository supports common core modules designed to equip students with
foundational AI literacy, data fluency, and essential human-centric digital skills.

---

## Target Audience
- Cross-disciplinary students
- No technical background assumed

---

## Core Themes
- AI in everyday life
- Human-in-the-loop decision-making
- Responsible and ethical AI
- Data-informed problem solving

---

## Modules Covered
- Data Fluency
- Artificial Intelligence and Its Impact
- Persuasive Communication with Data Storytelling
- Problem Solving with Creative and Computational Thinking
